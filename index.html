About MadeOf

----------------------------------------------------------------------

MadeOf is a lightweight 2D physics engine created in Java. It is designed to make adding realistic physics to small Java projects simple and fun.

What it does

MadeOf simulates basic physics behaviors such as gravity, collisions, and friction. You can create balls that move naturally, collide, and interact with each other, or draggable objects that users can manipulate with the mouse.

Why it exists

The engine was built as a personal learning project to understand how physics simulations work in Java. It’s a playful tool for experimenting with motion, collisions, and interactive elements.

Who it's for

MadeOf is perfect for hobby projects, educational purposes, or anyone curious about physics programming in Java. It’s simple to integrate, and you can modify it freely to explore new physics concepts.
